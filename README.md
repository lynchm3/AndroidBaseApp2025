# Android Base App
Base app app to be used as starting point to quickly kickstart projects

# Libraries:
- UI - Compose
- Dependency Injection - Hilt
- API Interface - Retrofit 2
- HTTP - OKHTTP
- Image Loading - Coil 3
- Coroutines
- Unit Tests - JUnit
- UI Tests - Espresso
- Mocking - Mockito
- Logging - Timber
